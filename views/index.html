Charge API

The Charge Service will be used by Lenders and Conveyancers to create Charges to be applied to the Land Register. There are two services that will be provided to Lenders and Conveyancers through which they can submit Charges.

  • An online transaction
  • This API

Expected process flow

1. Lender creates a Charge POST /charge

The lender uses the Charge API to create a Charge. They provide information on the mortgage product to use, the borrowers to list on the Charge, and the property to apply the Charge against.
